SCAMGUARD MVP - INSTALLATION GUIDE

1. Download all files into one folder
2. Open Chrome/Edge/Brave browser
3. Go to: chrome://extensions/
4. Enable "Developer mode" (toggle top-right)
5. Click "Load unpacked"
6. Select the scamguard-extension folder
7. Extension will install automatically

HOW TO TEST:
1. Visit a suspicious-looking page
2. Click the ScamGuard icon in toolbar
3. See risk analysis and warnings

PATTERNS DETECTED:
- Urgency phrases ("act now", "limited time")
- Payment pressure ("wire transfer", "gift cards")
- Deception ("you've won", "free money")
- Impersonation ("Microsoft support", "IRS official")

PERMISSIONS EXPLAINED:
- activeTab: Only reads current tab you're viewing
- scripting: Injects detection script when needed
- notifications: Shows alerts for high-risk pages

No data is collected. All analysis happens locally.
